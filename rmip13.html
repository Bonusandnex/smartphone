  • CPU : Apple : A15 Bionic Hexa Core (2+4)
  • ความเร็ว : 3.1 GHz
  • GPU : Apple GPU (4-core graphics) ,
    Apple Neural Engine (ANE) : 16‑core
  • RAM 6GB, ROM 128/256/512GB
